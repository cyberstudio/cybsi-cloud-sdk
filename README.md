Cybsi Cloud SDK
---------------
Python SDK для лёгкой интеграции с сервисами Cybsi Cloud (https://cybsi.cloud). Cybsi Cloud предоставляет данные о киберугрозах.

Библиотека имеет как синхронный, так и асинхронный интерфейс.
